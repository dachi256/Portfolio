    Interactive Projects

    Lord of the Maze

    The project is a Lord of the Rings-inspired maze game where players guide Frodo to Mount Doom, avoiding orcs and collecting the ring, while utilizing strategic gameplay and nostalgic visuals. The game employs object-oriented programming with features like collision detection, sound integration, and dynamic visual effects, enhancing the immersive experience.

    ThumbWave Studio

    The project combines interactive art with gesture control using ML5 and Arduino, allowing users to create dynamic visual art by manipulating sine wave structures through natural hand gestures. This setup is particularly beneficial for educational purposes, allowing users to interact with mathematical models through intuitive gestures, fostering a deeper understanding of complex concepts.

    Be Not Afraid

    The project, inspired by the biblical Ophanim, features a visual and auditory depiction of the celestial being. The functionality is mainly provided by WEBGL, attractors, and various animations. It includes rotating rings with numerous eyes, dynamic lighting, and sound effects to create an immersive experience.

    Astro Man

    AstroMan is a space-themed VR project featuring grappling hook mechanics in a cosmic environment. It combines physics-based movement with immersive visuals, including interactive celestial bodies and a dynamic space backdrop, to create a unique exploration experience.

    Zona

    This project crafts an atmospheric virtual environment drawing from horror aesthetics and film influences. It utilizes interactive audio-visual design to create an immersive, unsettling experience for first-person exploration.

    Bloodheart

    This project creates an immersive low-poly VR adventure environment with medieval elements. It features interactive weapons, drawing mechanics, and atmospheric audio-visual design. The world serves as a potential epilogue for a larger narrative-driven experience.

    Doctor Strange Demo

    This project uses a Rokoko motion capture suit to animate a Doctor Strange-inspired scene in Unity. It translates real-time body movements into fluid magical gestures, creating an immersive experience where users can manipulate reality like the Sorcerer Supreme.
